Alerts: Code

  • Version 1.0
  • Last updated: [Date]

Alerts are used to call attention to information that needs to stand out from the rest of the page content. Use alerts to provide contextual feedback messages for typical user actions.

HTML code 
<section class="alert alert-infoalert alert-successalert alert-warningalert alert-danger">
<p class="h3 mrgn-tp-0">Descriptive title</p><span class="wb-inv">Alert: InfoSuccessWarningDanger</span>
<p>Content of your alert <a href="#">link text</a>.</p>
</section>

Coding considerations:

If the alert doesn't have a title, add invisible text inside a <span class="wb-inv"> to provide the same experience to screen reader users
  • Begin <section> with a heading that is specific to the featured content, and contained within it
  • Do not use <section> within another section, or validation issues occur
    • Use <div> to style the alert
