Customisation

Option Description How to use
Full width Make a button occupy the full width of its column. Add the class btn-block to button.
Grouped buttons Buttons can be grouped side by side, such as for toggle buttons. Surround the grouped buttons with a div with the class btn-group.
Change sizing Buttons can be sized to improve visibility for call to actions or to save space in designs such as a toggle. Apply the following classes to the button:
  • Large: btn-lg
  • Small: btn-sm
  • Extra small: btn-xs
Call a javascript function Call a javascript function to complete the action. ((Would we give options? Leave javascript in its own separate guide?))
