Customisation

Option Description How to use
Left align card Used to align text to the left on cards, so it shows under the table heading. Meant to be applied item to each td you want to align left. Wrap the content you want to align left in a span with the class mobile-left.
Tablet view cards When table content is crowded on medium screens, this class createss the card view for tablets. Apply the class tblt-crds to the table.
No text break Stops text from breaking into another line, such as for a phone number or date. Apply the class nobreak to the td.
Small text Decreases the font size of headings and data in desktop view to fit lengthly content more compactly Apply the class txt-sm to the table.
Hide the caption When a relevant header is already above the table, the caption can be hidden Apply the class wb-inv to the caption.
Enable filtering Allows the table data to be searched Apply the class wb-tables to the table.
