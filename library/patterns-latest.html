Latest news: Government of Canada design system

Features the most recent news items posted by the institution, organization or entity.

On this page

  • When to use
  • What to avoid
  • How to implement
  • Discussion

When to use this pattern

Use this pattern on the following templates:

  • institutional landing page
  • ministerial profile

What to avoid

Make sure this pattern doesn't get in the way of top task content.

How to implement

You can use either the beta or the stable version of this pattern.

Beta Latest news pattern

The beta version of this pattern may still be subject to minor changes.

It's used in the beta version of the Institutional landing page template.

The beta version of this pattern will eventually replace the stable version.

Stable Latest news pattern

The stable version will eventually be replaced by the beta version.

Stable Latest news pattern guidance
  • It features the most recent news items posted by the entity
  • The heading is labelled “Latest”
  • 2 of the featured news items must include an image, a headline hyperlink, additional information and a description text:
    • can include promotional content, and be prioritized at the institution’s discretion
    • keep description text short and concise
    • if more than 2 images are required, images may be displayed in carousel format
    • if using the carousel format refer to Carousels in Canada.ca design pattern
  • The types of news products that must be listed as text-only are limited to only those available through news.gc.ca:
    • media advisories
    • backgrounders
    • news releases
    • speeches
    • statements
  • Consult the Canada.ca GitHub page for details on image sizing
